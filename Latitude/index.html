Latitude Analysis Dashboard

Where we see if latitude has an effect on weather elements.

What does this project hope to accomplish?

To analyze how weather elements change as you get away from the equator. We used data pulled from OpenWeatherMap API to gather a dataset on over 500 cities

After assembling the data, we used Matplotlib to plot latitude vs various weather elements. We looked into how latitude played an effect on temperature, humidity, cloudiness, and wind speed. This site provides visualizations and a summary of trends and correlations witnessed.
